# 数独 · Mobile Sudoku

一款简洁流畅的 iOS 数独游戏，使用 SwiftUI 原生开发。

## 功能

### 游戏玩法
- **四档难度**：Easy / Medium / Hard / Expert，线索数量从 45 递减至 25
- **候选数字（铅笔标记）**：在格子上标注候选数，冲突候选会触发动画提示
- **错误高亮**：填入错误数字时显示红色
- **提示**：每局限定次数，直接揭示正确答案
- **撤销**：最多回退 30 步，恢复棋盘和候选，不回退错误次数

### 游戏体验
- **暂停**：隐藏棋盘、停止计时；切换到后台自动暂停
- **错误次数上限**：首页可设置（1–10 次或无限制），达到上限显示游戏结束画面
- **继续游戏**：退出后返回首页，选同一难度可选择继续上局
- **通关庆祝**：通关时显示全屏动画覆盖层，破个人纪录时有专属金色祝贺界面

### 排行榜
- 每个难度独立保存前 10 名成绩
- 每条记录包含通关用时和错误次数
- 首页奖杯按钮和通关后均可查看

### 界面
- 深色 / 浅色 / 跟随系统 三档主题切换（长期保存）
- 适配 iPhone 16 Pro / Pro Max 全面屏

## 技术栈

- **语言**：Swift 5.9
- **框架**：SwiftUI（iOS 17.0+）
- **架构**：MVVM，纯值类型 Model
- **项目管理**：[XcodeGen](https://github.com/yonaskolb/XcodeGen)
- **持久化**：UserDefaults（排行榜、设置偏好）
- **无第三方依赖**

## 算法

| 模块 | 说明 |
|---|---|
| 生成器 | 随机回溯填满网格，随机挖空；每次挖空前用 solver 验证唯一解 |
| 求解器 | 标准回溯，`hasUniqueSolution` 计数至 2 时提前终止 |
| 错误检测 | 每次输入后对照答案标红，候选冲突检测同行/列/宫 |
| 异步生成 | Expert 等耗时较长的难度在后台线程生成，主线程不阻塞 |

## 构建

```bash
# 安装 XcodeGen（仅首次）
brew install xcodegen

# 生成 Xcode 项目
xcodegen generate

# 打开项目
open MobileSudoku.xcodeproj
```

在 Xcode 中选择目标设备，⌘R 运行。

**最低部署目标**：iOS 17.0
